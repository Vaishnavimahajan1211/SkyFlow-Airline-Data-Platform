# ✈️ SkyFlow - Enterprise Airline Data Platform

##  Project Overview

SkyFlow is an end-to-end Data Engineering project that simulates how an airline company collects, processes, transforms, and analyzes data from multiple systems.

The project will use:
- Python
- PySpark
- MySQL
- Apache Airflow
- Apache Kafka
- Power BI
- Docker
- Git & GitHub

##  Project Goal

Build a production-style Airline Data Platform capable of:

- Processing booking data
- Managing flight information
- Tracking passenger data
- Performing ETL pipelines
- Generating business analytics
- Supporting batch and streaming data processing

 Project Status: In Progress
